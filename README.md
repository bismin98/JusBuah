# Fruit Juice Website üçä

Website penjualan jus buah segar yang dibuat dengan Next.js dan Tailwind CSS.

## Fitur

- **Header dengan Logo Buah**: Menggunakan emoji buah jeruk sebagai logo
- **Menu Navigasi**: Tentang, Produk, dan Kontak
- **Background Buah Segar**: Menggunakan gambar dari Unsplash
- **Desain Colorful**: Tema buah-buahan dengan gradasi warna cerah
- **Responsive Design**: Tampilan optimal di semua perangkat

## Teknologi

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser di [http://localhost:3000](http://localhost:3000)

## Build untuk Production

```bash
npm run build
npm start
```

## Struktur Proyek

- `/app` - Aplikasi Next.js dengan App Router
- `/app/page.tsx` - Homepage utama
- `/app/layout.tsx` - Layout aplikasi
- `/app/globals.css` - Global styles dengan Tailwind CSS
